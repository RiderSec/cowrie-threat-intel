# Live 48h Threat Intelligence Capture Pipeline

## TLDR

Honeypot deployed on a public VPS to capture real world attack data, using a local VM using ELK stack to visualize the data cleanly, as well as a simple python script to convert the data to STIX and opentaxii as a TAXII server to provide IOCs.
